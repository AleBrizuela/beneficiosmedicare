Medigap Data Scraper

Scrapes Medicare.gov's public API for all 50 states + DC

How to use:
  1. Open this file in Chrome (must be on your computer, not a VM)
  2. Click "Start Scraping" below
  3. Wait ~3-5 minutes for all states to complete
  4. Click "Download All Data" to get the JS files
  5. Place the downloaded files in the data/ folder of the repo
